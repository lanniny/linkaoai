## 版本 A · 坦诚版（推荐首发）

```
大二的我决定干件事：
做一个叫"临考"的 AI 考试复习工具，主攻高数 / 线代 / 概率论。
30 天后向大家汇报结果。
先记一笔，免得我中途反悔。
```

适用：朋友圈首发 / 即刻
气质：低调真诚，不吹牛
副作用：低，几乎零风险

## 版本 B · 故事版（适合公众号 / 小红书首贴）

```
高数大一挂过一次。
那次复习像在迷雾里乱抓——课件一堆，时间一周，没人告诉我先看哪。

我决定做一个工具：
- 把课件喂进 AI
- 它吐出考点大纲（必考 / 重点 / 了解 三级）
- 给我针对性练习
- 按距离考试的天数倒排冲刺计划

名字叫"临考"。

7 天 MVP，30 天看真实数据。
有期末挂科焦虑的同学，6 月份可以来试试。
做不出来我直接发"惨败复盘"。
```

适用：长文场景，建立信任
气质：有故事、有用户视角
副作用：信息密度高，朋友圈会被截断

## 评论区话术（提前准备）

被问"用得上吗?" / "靠谱吗?"：
> "6 月期末月用得上的扣 1，我加你内测群"

被问"是不是替我写作业的工具?"：
> "不是。是辅助复习的，AI 帮你把课件转成大纲和练习题，所有内容仍以教材老师讲义为准。"

被问"和 ChatGPT 有啥区别?"：
> "ChatGPT 你得自己输入问题。临考是你扔课件和真题，它给你出考点大纲 + 针对性出题 + 距考天数的冲刺计划。一次配置，每天用。"

被问"你能做出来吗?"：
> "7 天 MVP 上线、30 天数据复盘。做不出来我会公开发惨败复盘，比假装做出来好。"

## 红线（写文案时务必避开）

- ❌ 避开学术诚信红线词：不出现"代·写" / "作·弊" / "包·过" / "保·过" / "100%·通过" 等任一字眼（详见 [day0-account-prerequisite.md](day0-account-prerequisite.md) 红线清单）
- ❌ 不晒同学的真实考试成绩当宣传素材
- ✅ 强调"辅助复习"、"提效工具"、"以教材为准"
